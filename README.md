## 📌 Sobre o Projeto

O **ExplicaAí!** é uma plataforma que conecta **alunos** que precisam de ajuda em matérias específicas com **tutores** voluntários que desejam compartilhar conhecimento e acumular horas complementares.

### 🎯 Problema que resolve

- Alunos com dificuldade sentem vergonha de pedir ajuda
- Alunos excelentes precisam de horas complementares, mas não há plataforma oficial
- Dificuldade em encontrar tutores por matéria específica

### ✨ Solução

Uma plataforma exclusiva para a escola que permite:
- Cadastro de alunos e tutores
- Busca por matéria
- Sistema de match entre aluno e tutor
- Agendamento de horários
- Avaliação de tutores com estrelas
- Agenda semanal de compromissos

## 🔧 Como Executar Localmente

### Pré-requisitos
- Navegador moderno (Chrome, Edge, Firefox, Safari)
- Git (opcional)

### Passos

1. **Clone o repositório**
git clone https://github.com/William-Matheus-SV/ExplicaAi-api.git

2. **Acesse a pasta**
cd ExplicaAi-api

3. **Abra o arquivo index.html no navegador**
    Clique duas vezes no arquivo
    Ou use uma extensão como Live Server no VS Code

4. **Faça login ou cadastre-se**

##Fluxo Básico
Login com uma das contas acima

Dashboard → veja seu perfil

Buscar → encontre tutores ou alunos por matéria

Perfil → visualize horários disponíveis

Match → selecione matéria e horário

Agenda → veja seus compromissos

Avaliação → avalie o tutor após o match
